# Portfolio

Personal site built with Astro, React, and Tailwind.

## Stack

- Astro 6 + React 19
- Tailwind CSS v4
- MDX for content
- Node adapter (standalone)

## Dev

```sh
pnpm install
pnpm dev
```

## Docker

```sh
cp .env.example .env
# fill in CONTACT_FORM_ACCESS_KEY
docker compose up -d
```
